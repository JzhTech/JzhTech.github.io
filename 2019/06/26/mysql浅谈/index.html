mysql浅谈

2019.06.26 Jzh 数据库  热度 ℃

下载mysql

  1. 进入官网下载 MySQL
  2. 进入官网，选择download
  3. 点击download后选择community社区版
  4. 选择windows版本的下载
  5. 点击进入安装
  6. 进入后页面最下方进行版本选择，此时是最新的版本，可以选择低版本
  7. 点击下载离线文件
  8. 进入该页面点击直接下载

安装mysql

  1. 点击安装程序
  2. 接受，下一步
  3. 选择custom自定义安装
  4. 根据图片提示下一步
  5. 安装
  6. next
  7. next
  8. next
  9. 默认端口号3306，直接next
  10. 设置密码，通常为root，next
  11. 下一步
  12. 安装
  13. finish
  14. next
  15. finish
  16. 找到数据库的安装位置，默认在如图
  17. 找到mysql文件夹
  18. 找到my.ini文件打开
  19. 根据图示修改数据库编码为utf8
  20. 修改后保存退出，打开服务，重启mysql服务
  21. 重启后搜索mysql找到他的命令窗口
  22. 进入命令窗口，输入密码自己设置的密码，这里是root，然后进入数据库
  23. 输入show variables like ‘character_set_%’;命令查看编码，为utf8则成功了。
